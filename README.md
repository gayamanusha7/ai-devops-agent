# AI DevOps Agent 🤖

## Overview
AI DevOps Agent is a secure AI-powered assistant that helps developers perform DevOps tasks using natural language.

Users authenticate using Auth0 and can interact with the agent through a web interface.

## Features
- Auth0 authentication
- AI DevOps assistant chat interface
- Secure user access
- Backend API agent logic

## Tech Stack
- React
- Node.js
- Express
- Auth0

## Architecture
User → Auth0 Login → React Frontend → Node Backend → DevOps Agent

## How to Run Locally

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
